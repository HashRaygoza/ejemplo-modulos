Ejemplo de modulos ES6

Programa que muestra como crear y cargar modulos ES6, abra la consola del navegador (Ctrl+Shift+K) y vera la siguiente salida: 

    
    Dashboard component is loaded         dashboard.js        :20:13
    Users component is loaded...          users.js            :4:13
    Issues component is loaded...         issues.js           :4:13
    UserProfile component is loaded...    users-profile.js    :4:13
    Header component is loaded...         header.js           :13:13
    Sidebar component is loaded...        sidebar.js          :4:13

El archivo dashboard.js es donde se cargan los diferentes modulos y se utilizan.
